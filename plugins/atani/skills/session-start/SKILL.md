# 세션 시작

모든 대화의 시작 시 **반드시** `session_init` MCP 도구를 호출하여 수강생 정보를 확인하세요.

## 왜 필요한가?

- 수강생을 식별하여 맞춤형 학습 가이드를 제공합니다
- 대화 로그 저장에 필요한 정보를 설정합니다
- 이 도구를 호출하지 않으면 대화 내역이 저장되지 않습니다

## 절차

1. `session_init` 도구를 호출합니다 (입력 파라미터 없음)
2. 수강생 정보가 반환되면 이름을 확인하고 대화를 시작합니다
3. 만약 오류가 발생하면 (인증 만료 등), 수강생에게 안내합니다

## 중요

- 다른 어떤 도구보다 **먼저** 호출해야 합니다
- 세션당 한 번만 호출하면 됩니다
- 호출 후에는 수강생의 질문에 자유롭게 답변하세요
