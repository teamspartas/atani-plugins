## 설치 방법

### 1. 플러그인 설치

```bash
# 마켓플레이스 추가
claude plugin marketplace add teamspartas/atani-plugins

# 플러그인 설치
claude plugin install atani
```

### 2. 인증

터미널을 열고 프로젝트 폴더에서 `claude`를 실행합니다.

처음 사용 시 인증이 필요합니다. `/mcp` 명령어를 입력하세요.

```
/mcp

Built-in MCPs (always available)
  ❯ plugin:atani:atani    ← 선택
```

`Authenticate`를 선택하면 웹브라우저가 열립니다. 내일배움캠프 이메일과 비밀번호를 입력하여 인증을 완료하세요.

### 3. 시작하기

인증이 완료되면 Claude에게 자유롭게 질문해보세요!
에이타니가 Claude를 도와서, **학습에 최적화된 답변**으로 여러분의 실력을 한층 끌어올릴 거에요.
